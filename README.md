---

# 📖 Project Overview

Understanding user listening behaviour is essential for music streaming platforms to improve engagement and personalise user experiences.

This project analyses Spotify listening history and transforms raw streaming data into meaningful business insights through an interactive Power BI dashboard.

The report enables users to explore listening trends across albums, artists, tracks, platforms, listening hours, and engagement metrics while using advanced DAX calculations and interactive report features.
